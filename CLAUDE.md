# Projekt: Hundezüchter-Website

## Projektbeschreibung
Website für eine Hundezüchterin zur Präsentation aktueller Würfe.
Interessenten können sich informieren, Kontakt aufnehmen und Welpen reservieren.

---

## Geplante Funktionen

### Seiten / Sektionen
1. **Aktueller Wurf** - Fotos, Namen, Geschlecht, Verfügbarkeit der Welpen
2. **Rasseinfo** - Charakter, Wesen, Haltungsanforderungen
3. **Ablauf & Bedingungen** - Bewerbungsprozess, Anforderungen an Käufer
4. **Kosten** - Preise transparent darstellen
5. **Kontakt / Reservierung** - Formular zum Melden
6. **FAQ-Chatbot** - Regelbasiert (kein API-Key nötig), beantwortet häufige Fragen

### Chatbot
- Regelbasierter FAQ-Bot (kein Claude API-Key, kostenlos)
- Beantwortet Fragen zu Rasse, Ablauf, Bedingungen, Kosten

---

## Technologie-Stack
- **Frontend:** Reines HTML / CSS / JavaScript (kein Framework)
- **Hosting:** Vercel oder Netlify (kostenloser Plan)
- **Repository:** GitHub
- **Domain:** Optional, später hinzufügbar (~10-15 €/Jahr bei IONOS, Strato o.ä.)

---

## Hosting-Plan
1. Website lokal bauen
2. Code auf GitHub hochladen
3. Vercel/Netlify mit GitHub verbinden → automatische Veröffentlichung
4. Kostenlose URL: `hundezucht-name.vercel.app`
5. Optional: Eigene Domain verknüpfen
